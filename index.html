BASIC CSS PRACTICE

FLEXBOX

Flexbox is a great skill to learn for styling elements on a webpage or within a container. When the display property is set to the value of 'flex' is will turn the container to a flex-container and children inside as flex-items. Flexbox allows developers to change the layout within a row or column.

flex-direction

The flex-direction property is used to tell the browser to layout the child elements horizontally or vertically.

By default declaring flex on a container will line the children up horizontally next to one another. If this is the goal you do not have to explictly declare the flex-direction. This means that justify-content is moving items across the main axis which is horizontal and align-items moves items veritcally against the cross axis. We can also reverse the order horizontally.
